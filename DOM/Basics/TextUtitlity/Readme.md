# 📝 Text Master - Smart Text Editor

Welcome to **Text Master**, a smart and stylish text editing web app built with **HTML**, **Bootstrap 5**, and **JavaScript DOM**. This tool is designed for effortless and efficient text manipulation and analysis. Whether you're drafting a note or analyzing sentence structure, Text Master has got you covered! 💪

---

## 🚀 Features

✨ **Live Preview**: Instantly view the entered text  
🔡 **Case Conversion**:  
&nbsp;&nbsp;&nbsp;&nbsp;🔹 Convert to UPPERCASE  
&nbsp;&nbsp;&nbsp;&nbsp;🔹 Convert to lowercase  

🧮 **Text Statistics** (On clicking **Add Text**):  
- 📚 **Total Words**  
- 🔢 **Total Characters**  
- ⏱️ **Estimated Reading Time**  

🧹 **Clear All**: Instantly clear the input and preview area

📱 **Responsive Design**: Built with Bootstrap 5 to ensure optimal experience across devices

---

## 🛠️ Technologies Used

- 🌐 **HTML5**
- 🎨 **Bootstrap 5**
- 🧠 **Vanilla JavaScript (DOM Manipulation)**

---

## ⚙️ How to Use

1. 💬 Type your text in the input box.
2. 🟡 Click **"Add Text"** to:
   - Display text in the **Preview** section
   - See word count, character count, and reading time
3. 🔵 Use **"Convert into Uppercase"** or 🟢 **"Convert into Lowercase"** to change text casing
4. 🔴 Click **"Clear All"** to reset everything

---

## 📊 Text Stats Calculation Logic

- **Words**: Split by spaces and new lines
- **Characters**: Includes all characters excluding leading/trailing spaces
- **Reading Time**: Estimated using average reading speed (~200 words per minute)
